# typescript demo project
## 使用项目
    npm i 或 cnpm i 或 yarn

## 项目初始化
    cnpm init
    cnpm i  typescript –save-dev
    tsc -init
    cnpm i ts-node --save-dev
    cnpm i @types/node --save-dev
    cnpm i nodemon --save-dev
    cnpm i rimraf --save-dev
    cnpm i cross-env --save-dev

## 项目运行调试

    npm run dev 运行
    npm run build 编译生成
    f5 调试

    带环境运行
    npm run start


    tsc -build // 简写 tsc -b
